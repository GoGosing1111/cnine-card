# v8.6.3 전투 연출 확장

- 전투 진행 시간을 약 7~10초로 확장
- 멤버 카드 5장이 한 장씩 순차 공격
- 카드 등급별 가상 스킬 연출 차등 적용
  - C~R: 기본 타격
  - SR~HR: 컬러 오라 및 강화 충격파
  - UR~SSR: 확대, 플래시, 버스트 연출
  - MA~FUR/LIMITED: 다색 오라, 슬로우, 대형 충격파
- 멤버팀 및 몬스터 HP 바 추가
- 서버에서 확정된 승패 결과에 맞춰 HP 연출 진행
- 몬스터 강공격/반격 연출 강화
- 몬스터 이미지, 구분, 이름, 전투력을 하나의 카드 프레임으로 통합
- 전투 판정, 보상, D1 구조 및 기존 API 변경 없음
