씨켓몬 와고 코인 지급 확장프로그램 V1.0.5

수정 내용
- 다른 닉네임 조회 확장프로그램이 추가한 [i] 문자열 제거
- 닉네임 뒤 회원번호(#숫자) 및 부가 표시 제거
- 중복 렌더링된 닉네임 자동 축약
- 원본 닉네임 DOM만 복제해 추출하고 보조 확장프로그램 요소는 제외
- 기존 CMS 인증 및 메시지 보상 지급 기능 유지

설치
1. 기존 확장프로그램을 제거합니다.
2. V1.0.5 ZIP을 압축 해제합니다.
3. chrome://extensions 에서 압축해제된 확장프로그램을 로드합니다.
4. 와고 페이지와 CMS 페이지를 새로고침합니다.
