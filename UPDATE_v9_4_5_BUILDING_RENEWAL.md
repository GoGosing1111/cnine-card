# v9.4.5 마을 건물 전면 리뉴얼

기준: `cnine-card-v9.4.4-tileset-map-renewal.zip`

## 변경 사항

- 제공된 `B_HouseExteriorTiles.png` 타일셋을 사용해 마을 건물 4종을 전면 재제작했습니다.
- 카드 상점: 붉은 지붕, 목재 외벽, 카드 간판 적용
- 카드 연구소: 녹색 지붕, 석재 외벽, 플라스크 간판 적용
- PVP 경기장: 파란 지붕, 강화 석재 외벽, 교차 검 간판 적용
- 모험 안내소: 황금 지붕, 벽돌 외벽, 별 간판 적용
- 각 건물에 처마, 용마루, 굴뚝, 창문, 화단, 외부 조명, 기초 그림자를 추가했습니다.
- 기존 건물 좌표, 출입구 판정, 내부 이동, 충돌 및 CMS 기능은 유지했습니다.

## 라이선스

원본 타일 에셋 제작자 표기와 라이선스 안내는 기존 포함 파일을 유지합니다.
