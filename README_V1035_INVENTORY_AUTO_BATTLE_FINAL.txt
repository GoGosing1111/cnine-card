인벤토리
- 헤더 메인 메뉴에 잘못 추가된 별도 인벤토리 탭 제거
- 컬렉션 옆 보라색 인벤토리 칸 복구
- 왼쪽 가방 아이콘, 인벤토리 제목, 보유 수량, 오른쪽 NEW 배지 구성
- CN VAULT / REWARD STORAGE 등 불필요한 영문 문구 제거
- 인벤토리 페이지 입장권·이벤트 탭 제거 상태 유지
- 인벤토리 페이지 로고 뒤 원형 장식 제거 및 로고만 표시
- 큐브 목록과 개봉 화면의 불필요한 영문 안내 제거
- 큐브 3종 이미지의 실제 체크무늬 배경을 제거하고 투명 PNG로 교체

PVE 자동전투
- 자동전투 선택 시 현재 남은 전투 횟수를 시작 시점 기준으로 고정
- 매 회차마다 기존 PVE 전투화면 전체 표시
- 카드 등장, 몬스터 등장, 공격, 반격, HP, 궁극기, 승패, 보상 연출 전부 재생
- 전투화면 상단에 자동전투 현재 회차 표시
- 각 회차는 기존 battle/fight API를 한 번씩 호출하여 개별 차감·보상 처리
- 결과 일괄 처리용 battle/auto API는 유저 자동전투 흐름에서 사용하지 않음
- 화면 클릭 시 자동전투 중단, 마지막 회차 후 누적 결과 표시
- 에너지 타이머가 자동전투 버튼 문구를 덮는 문제 수정

v1034 포함
- 궁극기 MP4 강제 음소거 제거
- 전투음이 켜져 있으면 MP4 내장 음향 재생
- 브라우저가 유음 재생을 차단할 때만 무음으로 영상 재시도

유저 화면 점검
- 레이드 결과의 CMS 문구 제거
- 점검 화면 관리자 접속 문구 제거
- 무제한 계정의 관리자·테스트 계정 표기 제거
