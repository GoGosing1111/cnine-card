# CNINE Card v5 업데이트

## 적용 기능
- 최근 획득 목록이 D1 서버 기록과 동기화되지 않던 문제 수정
- 중복 카드 획득 시 통합 `카드 조각` 지급
  - C 1 / U 2 / R 4 / SR 8 / HR 15 / UR 30 / SSR 60 / MA 120 / FUR 250
- 동일 카드 재료 없이 카드 조각만 사용하는 돌파 시스템
- SR 이상 카드 돌파 가능, 최대 ★10
- 단계별 소모량: 50, 100, 200, 350, 550, 800, 1100, 1450, 1850, 2300
- ★1~★10 단계별 카드 테두리, 광택, 네온, 번개, 불꽃, 오라, 파티클, LEGEND 효과
- 돌파 단계에 따른 카드 점수 보너스
- 카드 상세창에서 보유 조각, 돌파 단계, 다음 비용 확인 및 즉시 돌파
- D1 기존 DB 자동 업그레이드

## 최근 획득 오류 원인
기존 API 프로필 응답에 뽑기 이력이 포함되지 않았고, 프런트가 브라우저에 남아 있던 예전 history만 유지해 서버 뽑기 후 최근 획득 목록이 갱신되지 않았습니다. 이제 D1 `draw_logs` 최근 30건을 프로필에 포함해 로그인, 새로고침, 뽑기 직후 모두 동일하게 표시합니다.
