# v9.1.0 SSR 천장 / PvE 카드 드롭

## SSR 천장
- 프리미엄팩(`premium`)과 리미티드팩(`pickup`)에 각각 독립 카운트 적용
- SSR 이상 획득 시 해당 팩 카운트 0으로 초기화
- 1~60회: CMS 카드팩 기본 확률
- 61~69회 SSR 확률: 10%, 15%, 20%, 25%, 30%, 35%, 40%, 45%, 50%
- 70회: SSR 확정
- 10/20연속 뽑기도 각 슬롯별로 순차 계산

## PvE 카드 드롭
- 몬스터 승리 시 카드 1장을 직접 획득할 수 있음(카드팩 지급 아님)
- 전투관리 CMS에서 전체 사용 여부와 기본 확률 설정
- 몬스터별 카드 획득 확률 별도 설정 가능
- 몬스터별 값이 비어 있으면 기본 확률 사용
- 활성 일반 카드 중 `draw_weight` 가중치로 1장 지급
- 중복 카드는 기존 규칙대로 카드 수량 증가 및 카드 조각 지급
