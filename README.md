# CNINE Card V2.2.7

## 변경 사항
- OWNER 전용 유저 개인키 재발급 기능 추가
- 새 개인키는 재발급 직후 한 번만 표시
- 기존 개인키 즉시 무효화
- 해당 유저의 기존 로그인 세션 모두 해제
- 재발급 작업을 admin_logs에 기록

## 사용 방법
관리자 페이지 → 유저 코인 지급 → 닉네임 검색 → 유저 선택 → 개인키 재발급

추가 DB 초기화는 필요 없습니다. 기존 프로젝트에 덮어쓴 뒤 GitHub에 커밋·푸시하세요.
