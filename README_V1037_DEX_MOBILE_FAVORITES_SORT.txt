CNINE CARD V1037 누적 패치

[누적 포함]
1. 관리자 세션 중복 로그인 시 CMS 세션이 끊기던 문제 수정
2. CMS 몬스터 관리 / 큐브 관리 독립 메뉴 분리
3. 유저 도감 UX 개선

[도감 개선]
- 멤버별 즐겨찾기 별 버튼
- 즐겨찾기 멤버 항상 상단 우선 정렬
- 즐겨찾기 멤버만 보기
- 기본 순서 / 멤버 이름순 / 보유 카드 많은순 / 수집률 높은순 / 전체 카드 많은순
- 검색, 등급, 정렬, 즐겨찾기 상태 브라우저 저장
- 기존 멤버별 앨범 접기 및 카드 상세 기능 유지
- 태블릿/모바일/소형 모바일 대응

[안전 원칙]
- 기존 테이블/데이터/API 구조 변경 없음
- DROP/RENAME/재생성/데이터 삭제 없음
- 기존 도감 틀을 제거하지 않고 확장 방식으로 적용
