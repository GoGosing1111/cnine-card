씨켓몬 와고 코인 지급 확장프로그램 설치

1. Chrome 주소창에 chrome://extensions 입력
2. 오른쪽 위 개발자 모드 ON
3. 압축해제된 확장 프로그램을 로드합니다 클릭
4. 이 chrome-extension 폴더 선택
5. 확장프로그램 아이콘 클릭 → 씨켓몬 CMS 열기
6. CMS에 OWNER 또는 코인 지급 권한이 있는 관리자 계정으로 로그인
7. 와고 페이지에서 닉네임 클릭 → 연결된 씨켓몬 계정 확인 → 코인 지급

주의
- 와고 닉네임과 씨켓몬 닉네임이 달라도 wago_verifications의 VERIFIED 기록으로 연결됩니다.
- 같은 게시글/댓글에 같은 사유로 중복 지급하면 서버가 차단합니다.
- 지급 시 coin_logs, admin_logs, wago_extension_reward_receipts에 기록됩니다.
