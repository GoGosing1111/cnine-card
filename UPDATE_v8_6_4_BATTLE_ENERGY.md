# CNINE CARD v8.6.4

## 전투 횟수 시스템
- 기본 최대 전투 횟수 10회
- 전투 1회당 1회 소모
- 최대치 미만일 때 기본 15분마다 1회 자동 충전
- 한국시간 매일 00:00에 기본 10회로 복구
- 전투 화면에 현재 횟수, 게이지, 다음 충전 카운트다운 표시
- 전투 시작 API에서 서버 차감 및 조건부 업데이트로 중복 클릭 방지
- OWNER/ADMIN 및 점검 테스트 계정 무제한 옵션

## CMS
전투관리 > 전투 횟수 설정에서 아래 항목을 관리합니다.
- 기능 사용/무제한
- 최대 보유 횟수
- 매일 00시 복구량
- 1회 충전 간격(분)
- 전투당 소모 횟수
- OWNER/ADMIN 무제한
- 점검 테스트 계정 무제한

## 시간 표시 수정
- datetime-local 입력값을 UTC로 재변환하지 않고 한국시간 입력값 그대로 저장
- CMS 입력 시간과 사용자 점검 화면 표시 시간 불일치 수정

## DB 안정성
- 기존 테이블 DROP/RENAME/재생성 없음
