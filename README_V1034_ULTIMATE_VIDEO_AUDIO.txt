CNINE CARD v1034 궁극기 MP4 음향 복구 패치

적용 파일
- index.html
- js/app.js

변경 사항
- 궁극기 MP4의 강제 음소거 속성 제거
- 전투음이 켜져 있으면 MP4 내장 음향을 원본 음량으로 재생
- 유저가 전투음을 꺼둔 경우에는 기존 설정대로 무음 재생
- 브라우저가 유음 자동재생을 차단하는 환경에서는 영상 중단 방지를 위해 무음으로 자동 재시도
- JS 캐시 버전을 v1034로 갱신

주의
