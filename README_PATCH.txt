CNINE PvP 시즌 점수 전용 매칭/점수 개편 패치

적용 파일:
- functions/api/[[path]].js
- js/app.js
- index.html

적용 내용:
1) 카드 총점 매칭 조건 제거
2) 시즌 점수 범위 단계 확장: ±200 → ±400 → ±700 → ±1200
3) 시즌 점수 차이가 가까운 순으로 최대 6명 표시
4) 점수 획득/차감:
   상대 +500 이상: 승 +36 / 패 -6
   상대 +200~499: 승 +30 / 패 -10
   ±199: 승 +24 / 패 -16
   상대 -200~-499: 승 +18 / 패 -20
   상대 -500 이하: 승 +12 / 패 -24
5) 상대 목록에 점수 차이 및 예상 승/패 점수 표시
6) OWNER/ADMIN/정지/본인 제외 및 동일 상대 반복 제한 유지

적용 방법:
프로젝트 루트에 압축을 풀어 동일 경로로 덮어쓰기 후 배포.
