CNINE LIMITED GRADE v9.4.7

프로젝트 루트에 폴더 구조 그대로 덮어쓰기.

- FUR 아래 LIMITED 신규 등급 추가
- limited_total이 설정된 기존 한정판 카드만 안전하게 LIMITED로 자동 전환
- 신규/수정 카드에서 한정 수량 입력 시 서버가 등급을 LIMITED로 강제
- 일반 카드가 LIMITED로 저장되는 것은 차단
- CMS 카드 목록/대기함/필터/등급 선택에 LIMITED 표시
- 유저 카드/도감/뽑기 결과에 LIMITED 표시 및 전용 프레임 적용
- 일반 FUR/MA 카드와 일반 뽑기 확률은 변경하지 않음
- DROP/RENAME/테이블 재생성/데이터 삭제 없음
