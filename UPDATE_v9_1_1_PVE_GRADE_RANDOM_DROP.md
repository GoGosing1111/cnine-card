# v9.1.1 PvE 등급 랜덤 카드 드롭 수정

- 몬스터별 카드 드롭 확률 입력 제거
- 전투 CMS에서 모든 몬스터 공통 카드 획득 확률 관리
- 카드 획득 성공 시 CMS의 등급별 확률로 등급을 먼저 추첨
- 선택된 등급의 공개·활성·일반 카드 중 1장을 균등 랜덤 지급
- 카드별 draw_weight와 개별 카드 설정은 사용하지 않음
- 한정 카드, 대기 카드, 비활성 카드는 드롭 대상에서 제외
- 등급 확률 합계가 100%가 아니면 저장 차단
- 기본 등급 확률: C 40 / U 25 / R 15 / SR 10 / HR 6 / UR 3 / SSR 1 / MA 0 / FUR 0
